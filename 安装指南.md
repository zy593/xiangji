# 📱 相机海报生成器 - 安装指南

## 🎯 快速开始

### 第一步：生成应用图标（必须）

应用需要图标文件才能安装。有两种方法：

#### 方法一：使用在线工具生成（推荐）
1. 访问 https://www.pwabuilder.com/imageGenerator
2. 上传一张图片或使用默认图标
3. 下载生成的图标文件
4. 将 `icon-192.png` 和 `icon-512.png` 放到项目文件夹中

#### 方法二：手动创建
1. 准备两张图片：
   - `icon-192.png` (192x192像素)
   - `icon-512.png` (512x512像素)
2. 将图片放到项目文件夹中（与index.html同一目录）

#### 方法三：使用提供的图标生成器
1. 打开 `icon-generator.html` 文件
2. 点击"生成图标"按钮
3. 点击"下载图标"按钮
4. 将下载的文件放到项目文件夹中

## 📲 手机安装步骤

### iPhone (iOS) - Safari浏览器

1. **打开应用**
   - 在iPhone的Safari浏览器中打开 `index.html` 文件
   - 或者通过服务器访问（需要HTTPS）

2. **添加到主屏幕**
   - 点击底部的"分享"按钮（方框+箭头图标）
   - 向下滚动，找到"添加到主屏幕"
   - 点击"添加到主屏幕"
   - 可以修改应用名称（默认是"相机海报"）
   - 点击右上角"添加"

3. **使用应用**
   - 在主屏幕找到应用图标
   - 点击图标打开应用
   - 应用会全屏运行，像原生APP一样

### Android - Chrome浏览器

1. **打开应用**
   - 在Chrome浏览器中打开 `index.html` 文件
   - 或者通过服务器访问（需要HTTPS）

2. **安装应用**
   - 方法一：浏览器会显示"添加到主屏幕"的提示，点击"安装"
   - 方法二：点击右上角菜单（三个点）
   - 选择"添加到主屏幕"或"安装应用"
   - 点击"安装"按钮

3. **使用应用**
   - 在主屏幕或应用抽屉中找到应用
   - 点击图标打开应用
   - 应用会全屏运行

## 💻 桌面浏览器安装

### Chrome / Edge

1. 在浏览器中打开应用
2. 地址栏右侧会显示"安装"图标（或"+"图标）
3. 点击"安装"按钮
4. 确认安装
5. 应用会以独立窗口运行

### Firefox

1. 在浏览器中打开应用
2. 点击地址栏的"+"图标
3. 选择"安装此站点为应用"
4. 确认安装

## 🌐 通过服务器访问（推荐）

### 为什么需要服务器？

- PWA功能需要HTTPS环境（localhost也可以）
- 更好的性能和体验
- 支持离线功能

### 方法一：使用本地服务器

#### Python（如果已安装）
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
然后在浏览器访问：`http://localhost:8000`

#### Node.js（如果已安装）
```bash
# 安装http-server
npm install -g http-server

# 运行服务器
http-server -p 8000
```
然后在浏览器访问：`http://localhost:8000`

#### VS Code Live Server
1. 安装VS Code的"Live Server"扩展
2. 右键点击 `index.html`
3. 选择"Open with Live Server"

### 方法二：上传到服务器

1. 将所有文件上传到支持HTTPS的服务器
2. 通过网址访问应用
3. 按照上面的步骤安装

## 🔧 常见问题

### Q: 为什么看不到"安装"选项？
A: 
- 确保图标文件已存在
- 确保在HTTPS或localhost环境下
- 尝试刷新页面
- 检查浏览器是否支持PWA

### Q: 安装后无法使用摄像头？
A:
- 检查浏览器权限设置
- 确保在HTTPS环境下（摄像头需要安全环境）
- 重新授予权限

### Q: 图标显示不正确？
A:
- 确保图标文件名称正确（icon-192.png, icon-512.png）
- 确保图标文件在正确的位置（与index.html同一目录）
- 清除浏览器缓存后重试

### Q: 如何卸载应用？
A:
- iOS: 长按图标，选择"删除App"
- Android: 长按图标，拖到卸载区域
- 桌面: 在浏览器设置中卸载
