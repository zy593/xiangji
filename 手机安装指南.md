# 📱 手机安装指南 - 完整版

## 🎯 安装方式总览

有两种方式可以安装到手机：

1. **PWA方式**（最简单，无需下载APK）
   - 直接在浏览器中访问网址
   - 添加到主屏幕
   - 像原生APP一样使用

2. **APK方式**（Android专用）
   - 打包成APK文件
   - 下载到手机安装
   - 完全像原生应用

---

## 方式一：PWA方式安装（推荐，最简单）

### ✅ 前提条件
- GitHub Pages已启用
- 获得网址：`https://zy593.github.io/xiangji/`

### 📱 Android手机安装步骤

1. **打开浏览器**
   - 使用Chrome浏览器（推荐）
   - 访问：`https://zy593.github.io/xiangji/`

2. **安装提示**
   - 浏览器会自动显示"添加到主屏幕"的提示
   - 点击"安装"按钮
   - 或者点击右上角菜单（三个点）→ "添加到主屏幕"

3. **完成安装**
   - 应用会出现在主屏幕
   - 点击图标即可使用

### 📱 iPhone安装步骤

1. **打开Safari浏览器**
   - 必须使用Safari（不能用Chrome）
   - 访问：`https://zy593.github.io/xiangji/`

2. **添加到主屏幕**
   - 点击底部"分享"按钮（方框+箭头图标）
   - 向下滚动，找到"添加到主屏幕"
   - 点击"添加到主屏幕"
   - 可以修改应用名称
   - 点击右上角"添加"

3. **完成安装**
   - 应用会出现在主屏幕
   - 点击图标即可使用

## 方式二：APK方式安装（Android专用）

### ✅ 前提条件
- GitHub Pages已启用
- 获得网址：`https://zy593.github.io/xiangji/`

### 第一步：打包APK

1. **访问PWABuilder**
   - 打开：https://www.pwabuilder.com/

2. **输入网址**
   - 在输入框中输入：`https://zy593.github.io/xiangji/`
   - 点击"Start"按钮

3. **等待分析**
   - 等待几秒钟，系统会分析你的应用

4. **生成APK**
   - 点击"Build My PWA"按钮
   - 选择"Android"平台
   - 点击"Generate Package"按钮
   - 等待打包完成（1-3分钟）

5. **下载APK**
   - 打包完成后，点击"Download"按钮
   - 下载APK文件到电脑

### 第二步：传输到手机

**方法1：USB传输**
- 用USB线连接手机和电脑
- 将APK文件复制到手机

**方法2：云盘/微信传输**
- 上传APK到云盘（百度网盘、OneDrive等）
- 或通过微信发送给自己
- 在手机上下载

**方法3：邮件传输**
- 将APK作为附件发送到自己的邮箱
- 在手机上打开邮件下载

### 第三步：安装APK

1. **允许安装未知来源**
   - 打开手机"设置"
   - 找到"安全"或"应用"设置
   - 开启"允许安装未知来源应用"
   - 或安装时点击"设置"允许

2. **安装APK**
   - 在手机文件管理器中找到APK文件
   - 点击APK文件
   - 点击"安装"按钮
   - 等待安装完成

3. **完成**
   - 应用已安装到手机
   - 可以在应用列表中找到

## ❓ 常见问题

### Q1: 访问网址显示404？
**A**: 
- 检查GitHub Pages是否已启用
- 等待几分钟后重试（首次部署需要时间）
- 确认网址正确：`https://zy593.github.io/xiangji/`

### Q2: 没有"添加到主屏幕"选项？
**A**: 
- Android：确保使用Chrome浏览器
- iPhone：确保使用Safari浏览器
- 检查是否在HTTPS环境下

### Q3: APK安装失败？
**A**: 
- 确保已允许"未知来源"安装
- 检查APK文件是否完整下载
- 尝试重新下载APK

### Q4: 应用无法使用相机？
**A**: 
- 首次使用时授予摄像头权限
- 在手机设置中检查应用权限

### Q5: iPhone无法安装？
**A**: 
- 必须使用Safari浏览器
- 不能使用Chrome或其他浏览器
- 确保网址是HTTPS
